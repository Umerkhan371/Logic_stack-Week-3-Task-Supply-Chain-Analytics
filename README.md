# 📦 Supply Chain Analytics Challenge 

## 📖 Project Overview

This project was completed as part of the **LogicStack Data Analysis Internship – July 2026 (Week 3 Task)**. The objective was to analyze a real-world supply chain dataset using **Python** for data cleaning and exploratory data analysis (EDA), and **Power BI** for creating an interactive business dashboard.

The project focuses on evaluating shipment performance, delivery efficiency, freight costs, vendor performance, and logistics trends to generate meaningful business insights.


## 🎯 Objectives

* Clean and preprocess raw supply chain data.
* Perform Exploratory Data Analysis (EDA).
* Analyze shipment and delivery performance.
* Evaluate freight cost and vendor efficiency.
* Build an interactive Power BI dashboard.
* Generate business insights and recommendations.

## 🛠️ Tools & Technologies

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Power BI**
* **Jupyter Notebook**

## 📊 Data Cleaning

The dataset was cleaned using Python by performing the following tasks:

* Converted date columns into proper datetime format.
* Identified and handled missing values.
* Converted numeric columns into appropriate data types.
* Removed inconsistencies and invalid records.
* Prepared the dataset for analysis and visualization.


## 📈 Exploratory Data Analysis (EDA)

The following analyses were performed:

### Shipment Analysis

* Most common shipment mode
* Total shipments by country
* Shipment distribution

### Delivery Performance

* Delivery delay calculation
* Average delivery time
* Delayed shipment identification

### Cost Analysis

* Total freight cost
* Average line item value
* Insurance cost analysis (where available)

### Group-Based Analysis

* Country vs Total Shipments
* Vendor vs Total Cost
* Product Group vs Total Value
* Shipment Mode vs Average Delay


## 📉 Visualizations

Python visualizations include:

* 📊 Bar Chart – Country vs Shipments
* 🥧 Pie Chart – Shipment Mode Distribution
* 📈 Line Chart – Delivery Trend Over Time

## 📊 Power BI Dashboard

### KPI Cards

* Total Shipments
* Total Freight Cost
* Average Delivery Delay
* Total Line Item Value

### Dashboard Visuals

* Country vs Total Shipments
* Delivery Trend Over Time
* Shipment Mode Distribution
* Vendor vs Total Cost

## 🔍 Key Insights

### Python Analysis

* Identified the most frequently used shipment mode.
* Measured delivery delays across shipments.
* Compared shipment volumes by country.
* Evaluated vendor-wise freight costs.
* Analyzed overall shipment trends over time.

### Power BI Dashboard

* Dashboard highlights shipment performance through interactive KPIs.
* Vendor and country comparisons improve operational visibility.
* Delivery trend analysis helps monitor logistics performance.

## 💡 Business Recommendations

* Improve logistics planning for routes with higher delivery delays.
* Optimize vendor selection based on cost efficiency and delivery performance.
